# Parametres entreprise

La page **Parametres entreprise** est le centre de configuration de votre organisation. Toutes les informations saisies ici apparaissent sur vos documents officiels : factures, devis, bulletins de paie, boutique en ligne.

## Acces

**Administration** > **Parametres entreprise**

## Informations legales

1. Allez dans **Administration** > **Parametres entreprise**.
2. Dans l'onglet **Entreprise**, renseignez les champs suivants :
   - **Nom de l'entreprise** : raison sociale (ex. : "Ma Societe SARL").
   - **Forme juridique** : selectionnez dans la liste (SARL, SA, EURL, EI, SNC, etc. -- cadre OHADA).
   - **RCCM** : numero au registre du commerce et du credit mobilier (ex. : BJ-COT-2024-B-1234).
   - **Capital social** : montant en chiffres (ex. : 1 000 000).
   - **Adresse** : siege social de l'entreprise.
   - **Telephone** : numero professionnel avec indicatif pays.
   - **Logo** : telechargez le logo de votre entreprise (image PNG ou JPG, max 2 Mo).
3. Cliquez sur **Enregistrer**.

## Informations fiscales

1. Dans la meme page, ouvrez l'onglet **Fiscalite**.
2. Renseignez les identifiants fiscaux :
   - **IFU** (Benin, Burkina Faso) : Identifiant Fiscal Unique.
   - **NIF** (Cote d'Ivoire, Togo) : Numero d'Identification Fiscale.
   - **NINEA** (Senegal) : Numero d'Identification Nationale des Entreprises.
   - **RCCM** : deja saisi dans l'onglet Entreprise.
   - **Numero CNSS employeur** : identifiant de la Caisse Nationale de Securite Sociale (si applicable).
3. Configurez les parametres de TVA :
   - **Taux de TVA par defaut** : 18% (Benin, Cote d'Ivoire, Senegal), 19.25% (Togo), ou autre selon le pays.
   - **Assujetti a la TVA** : cochez si votre entreprise est assujettie.
4. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/administration/parametres-entreprise-fiscalite.png" alt="Informations fiscales de l'entreprise" />

## Parametres de facturation

1. Ouvrez l'onglet **Facturation**.
2. Configurez les options suivantes :
   - **Mode de facturation** : choisissez entre **Simple** (factures sans certification), **Normalisee** (certification e-MECeF/MCF), ou **Les deux**.
   - **Message pied de page** : texte qui apparait en bas de chaque facture (ex. : "Sauf erreurs et omissions -- TVA non applicable selon article 259 B du CGI").
   - **Delai de paiement** : nombre de jours avant echeance (ex. : 30 jours).
   - **Penalites de retard** : texte mentionnant les penalites applicables en cas de retard de paiement.
3. Telechargez votre **signature** ou **cachet de l'organisation** : image PNG (max 2 Mo) qui sera apposee sur les documents certifies.
4. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/administration/parametres-entreprise-facturation.png" alt="Parametres de facturation" />

## Options comptables

1. Ouvrez l'onglet **Comptabilite**.
2. Activez la **comptabilite automatique** si vous souhaitez que les ecritures comptables soient generees automatiquement a chaque facture, paiement ou mouvement de stock.
3. Selectionnez le **plan comptable** adapte a votre pays (SYSCOHADA pour les pays OHADA).
4. Configurez les **journaux** par defaut : ventes, achats, banque, caisse, operations diverses.
5. Cliquez sur **Enregistrer**.

## Bon a savoir

- Toutes les informations saisies dans les parametres entreprise peuvent etre modifiees a tout moment.
- Les modifications du logo, de la signature ou des mentions legales sont appliquees immediatement sur les nouveaux documents.
- Les documents deja edites conservent les informations qui etaient en vigueur au moment de leur creation.
