# Collaborateurs et rôles

Gérez qui accède à quoi dans votre entreprise.

## Inviter un collaborateur

**Paramètres** → **Collaborateurs** → **Inviter**

## Rôles prédéfinis

| Rôle | Description |
|------|-------------|
| Propriétaire | Accès total |
| Commercial | CRM et ventes |
| Comptable | Finance et facturation |
| Responsable RH | Paie et employés |
| Manager | Lecture/écriture sans suppression |
| Collaborateur | Accès limité |

## Permissions

Chaque rôle définit des droits par module : **lecture**, **écriture**, **suppression**.

Le propriétaire peut créer des **rôles personnalisés** dans **Paramètres** → **Rôles & permissions**.
