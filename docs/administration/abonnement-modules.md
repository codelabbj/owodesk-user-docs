# Abonnement et modules

Le propriétaire gère l'abonnement OwoDesk et active les modules.

## Où accéder

**Abonnement** dans le menu (réservé au propriétaire)

## Plans

| Plan | Modules inclus |
|------|----------------|
| Starter | CRM, facturation, stock, RH, compta, POS, projets, demandes… |
| Pro | Starter + e-commerce |
| Business / Enterprise | Tous les modules plateforme |

## Activer un module

1. Ouvrir **Abonnement**
2. Choisir le module à activer
3. Sélectionner la formule (mensuelle / annuelle)
4. Procéder au paiement

:::info À compléter
Mettre à jour selon les offres commerciales en vigueur.
:::
