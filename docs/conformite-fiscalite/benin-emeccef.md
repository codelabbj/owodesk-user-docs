# e-MECeF — Bénin (référence)

Le **e-MECeF** (système électronique de facturation normalisée) est le dispositif de la Direction Générale des Impôts du Bénin pour certifier les factures et tickets de vente.

## Obligation

Les entreprises assujetties doivent émettre des **factures normalisées** avec code et QR code vérifiables.

## Dans OwoDesk

OwoDesk couvre :

- La **configuration** fiscale (adaptateur API ou machine MCF)
- La **certification des factures** à la finalisation
- La **certification automatique des tickets POS** à l'encaissement

## IFU et NIM

- **IFU** : identifiant fiscal de votre entreprise — **Paramètres** → **Entreprise**
- **NIM** : numéro d'identification mécanographique du système de facturation d'entreprise (SFE)

Ces identifiants doivent correspondre à vos enregistrements officiels.
