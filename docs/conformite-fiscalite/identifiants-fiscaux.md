# Identifiants fiscaux par pays

Chaque pays utilise son propre identifiant fiscal pour les entreprises.

| Pays | Identifiant | Exemple de libellé |
|------|-------------|-------------------|
| Bénin | IFU | Identifiant Fiscal Unique |
| Côte d'Ivoire | NIF | Numéro d'Identification Fiscale |
| Sénégal | NINEA | Numéro d'Identification Nationale des Entreprises |

## Où le renseigner

**Paramètres** → **Entreprise** → champ identifiant fiscal

:::tip
Un identifiant correct est indispensable pour la certification e-MECeF au Bénin.
:::
