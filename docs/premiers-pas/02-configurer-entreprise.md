# Configurer son entreprise

Apres avoir [cree votre compte](/premiers-pas/creer-compte) et vous etre connecte, l'assistant de premiere configuration (`/app/onboarding`) s'affiche automatiquement tant que l'onboarding n'est pas termine. Il permet de completer votre profil, les informations de votre entreprise et d'inviter des collaborateurs.

Cet assistant concerne le **proprietaire** du compte (createur de l'organisation). Si vous rejoignez une entreprise par invitation, seule l'etape **Votre profil** vous est proposee.

## Procedure

1. Connectez-vous sur [owo.bj](https://owo.bj). Si votre configuration initiale n'est pas terminee, vous etes redirige vers l'assistant d'onboarding.
2. **Etape 1 -- Votre profil** :
   - Renseignez votre **nom complet** (deja pre-rempli si saisi a l'inscription).
   - Ajoutez votre **telephone personnel** (facultatif).
   - Telechargez une **photo de profil** (facultatif, max 2 Mo).
   - Cliquez sur **Continuer**.
3. **Etape 2 -- Votre entreprise** : completez les informations de votre societe, puis cliquez sur **Continuer**.
   - **Identite** :
     - **Nom de l'entreprise** (obligatoire, pre-rempli depuis l'inscription).
     - **Forme juridique** (facultatif) : SA, SARL, SAS, EURL, EI, etc.
     - **RCCM** (facultatif).
     - **Capital social** (facultatif).
     - **Logo de l'entreprise** (facultatif, max 2 Mo).
   - **Localisation** :
     - **Pays** (pre-rempli depuis l'inscription).
     - **Devise** : renseignee automatiquement selon le pays (ex. XOF pour le Benin, la Cote d'Ivoire, le Senegal).
     - **Fuseau horaire** : propose par defaut selon le pays, modifiable si besoin.
     - **Telephone de l'entreprise** (facultatif).
     - **Adresse** (facultatif).
   - **Fiscal** :
     - **Identifiant fiscal** : libelle adapte au pays (IFU, NIF, NINEA, etc.).
     - **Mode de facturation** : facture simple uniquement, facture normalisee uniquement, ou les deux.
4. **Etape 3 -- Collaborateurs** (facultatif) :
   - Saisissez l'**email** du collaborateur (obligatoire).
   - Ajoutez son **nom** (facultatif).
   - Selectionnez un **role** dans la liste (obligatoire) : Commercial, Comptable, Responsable RH, Manager, Collaborateur, etc. Le role Proprietaire n'est pas proposable ici.
   - Cliquez sur **Inviter**. Repetez pour chaque collaborateur.
   - Les invitations envoyees apparaissent dans la liste sous le formulaire.
   - Cliquez sur **Acceder au tableau de bord** pour terminer, ou sur **Passer cette etape** pour ignorer les invitations pour l'instant.

## Bon a savoir

- Le nom et le pays de l'entreprise sont deja connus depuis l'inscription : l'assistant sert surtout a les completer et a ajouter les informations legales et fiscales.
- Vous pouvez inviter d'autres collaborateurs plus tard depuis **Parametres** > **Collaborateurs**.
- Si vous etes invite dans une organisation existante, l'assistant ne comporte qu'une etape profil avant l'acces au tableau de bord.

## Modifier les informations plus tard

Les informations saisies restent modifiables depuis **Parametres** (icone en bas du menu lateral) :
- **Profil personnel** : nom, telephone, photo.
- **Entreprise** : identite, localisation, fiscal.
- **Collaborateurs** : invitations et membres de l'equipe.
