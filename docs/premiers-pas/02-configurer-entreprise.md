# Configurer votre entreprise (onboarding)

Après avoir créé votre compte et validé votre email, OwoDesk vous guide dans un **assistant de première configuration** en **3 étapes**.

Cet assistant s'affiche automatiquement à la première connexion. Vous pouvez aussi y revenir tant qu'il n'est pas terminé — OwoDesk vous y redirige si nécessaire.

:::info Qui voit quoi ?
- **Propriétaire** (celui qui a créé le compte) : les **3 étapes** ci-dessous.
- **Collaborateur invité** : uniquement l'étape **Votre profil**, puis accès direct à l'application.
:::

Une barre de progression en haut et un menu à gauche indiquent où vous en êtes. Cliquez sur **Continuer** à chaque étape pour avancer.

---

## Étape 1 — Profil du propriétaire

**Titre à l'écran :** *Votre profil personnel*

Ces informations concernent **vous**, en tant que dirigeant ou responsable du compte — pas encore l'entreprise.

| Champ | Obligatoire | Description |
|-------|-------------|-------------|
| **Nom complet** | Oui | Votre nom tel qu'il apparaîtra dans l'application (ex. : Jean Dupont). Minimum 2 caractères. |
| **Téléphone personnel** | Non | Votre numéro mobile ou fixe. L'indicatif pays est proposé automatiquement. |
| **Photo de profil** | Non | Cliquez ou glissez une image (max. 2 Mo). Elle s'affiche sur votre compte utilisateur. |

Cliquez sur **Continuer** pour enregistrer et passer à l'étape suivante.

## Étape 2 — Informations de l'entreprise

**Titre à l'écran :** *Votre entreprise*

Complétez l'identité légale et les coordonnées de votre société. Ces données apparaîtront sur vos **factures**, documents officiels et **boutique en ligne**.

### Identité

| Champ | Obligatoire | Description |
|-------|-------------|-------------|
| **Nom de l'entreprise** | Oui | Raison sociale (ex. : Ma Société SARL). |
| **Forme juridique** | Non | SARL, SA, SAS, EURL, entreprise individuelle, etc. (cadre OHADA). |
| **RCCM** | Non | Numéro au registre du commerce (ex. : BJ-COT-2024-B-1234). |
| **Capital social** | Non | Montant du capital en chiffres. |
| **Logo** | Non | Image de votre entreprise (max. 2 Mo). Utilisée sur les factures et la vitrine e-commerce. |

### Localisation

| Champ | Description |
|-------|-------------|
| **Pays** | Pays d'implantation de l'entreprise. |
| **Devise** | Renseignée automatiquement selon le pays (ex. : **XOF** pour le Bénin, la Côte d'Ivoire, le Sénégal…). |
| **Fuseau horaire** | Ajusté selon le pays (ex. : Porto-Novo pour le Bénin). Modifiable si besoin. |
| **Téléphone de l'entreprise** | Numéro professionnel. L'indicatif suit le pays choisi. |
| **Adresse** | Adresse postale complète (ex. : 123, Avenue principale, Cotonou). |

### Fiscal

| Champ | Description |
|-------|-------------|
| **Identifiant fiscal** | Le libellé change selon le pays : **IFU** (Bénin, Burkina Faso), **NIF** (Côte d'Ivoire, Togo…), **NINEA** (Sénégal). |
| **Mode de facturation** | Choisissez comment vous facturez : facture simple, facture normalisée (e-MECeF au Bénin), ou les deux. |

Cliquez sur **Continuer** pour enregistrer et passer à l'invitation des collaborateurs.

<DocImage src="/img/guides/premiers-pas/onboarding-entreprise.png" alt="Étape 2 — Informations de l'entreprise" />

:::tip Modifier plus tard
Toutes ces informations restent modifiables dans **Paramètres** → **Entreprise**, même après l'onboarding.
:::

## Étape 3 — Inviter des collaborateurs

**Titre à l'écran :** *Invitez des collaborateurs*

Ajoutez les membres de votre équipe. Chaque personne reçoit un **email d'invitation** pour rejoindre votre espace OwoDesk.

| Champ | Obligatoire | Description |
|-------|-------------|-------------|
| **Email** | Oui | Adresse professionnelle du collaborateur. |
| **Nom** | Non | Prénom et nom (facultatif, pour personnaliser l'invitation). |
| **Rôle** | Oui | Définit ce que la personne peut voir et faire (Commercial, Comptable, Responsable RH, etc.). Le rôle *Propriétaire* n'est pas proposé ici. |

Cliquez sur **Inviter** pour envoyer. Vous pouvez inviter **plusieurs personnes** l'une après l'autre ; la liste des invitations envoyées s'affiche en dessous.

### Terminer l'onboarding

Deux options :

- **Accéder au tableau de bord** — si vous avez invité votre équipe (ou souhaitez le faire plus tard).
- **Passer cette étape** — si vous préférez travailler seul pour l'instant. Vous pourrez inviter des collaborateurs depuis **Paramètres** → **Collaborateurs**.

## Après l'onboarding

Une fois terminé, vous arrivez sur le **tableau de bord**. Un bandeau peut vous proposer d'**activer vos premiers modules** (CRM, facturation, stock…) selon votre abonnement.
