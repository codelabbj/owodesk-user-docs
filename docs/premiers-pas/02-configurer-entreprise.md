# Configurer son entreprise

Apres avoir cree votre compte et valide votre email, l'assistant de premiere configuration vous guide pour configurer votre entreprise.

## Procedure

1. Connectez-vous a OwoDesk. L'assistant de configuration s'affiche automatiquement.
2. **Etape 1 -- Profil du proprietaire** :
   - Renseignez votre **nom complet**.
   - Ajoutez votre **telephone personnel** (facultatif).
   - Telechargez une **photo de profil** (facultatif, max 2 Mo).
   - Cliquez sur **Continuer**.
3. **Etape 2 -- Informations de l'entreprise** :
   - Saisissez le **nom de votre entreprise** (raison sociale).
   - Selectionnez le **pays** d'implantation.
   - La **devise** est renseignee automatiquement (XOF pour le Benin, la Cote d'Ivoire, le Senegal).
   - Remplissez l'**identifiant fiscal** (IFU pour le Benin, NIF pour la Cote d'Ivoire, NINEA pour le Senegal).
   - Choisissez le **mode de facturation** : facture simple, facture normalisee, ou les deux.
   - Cliquez sur **Continuer**.
4. **Etape 3 -- Inviter des collaborateurs** (facultatif) :
   - Saisissez l'**email** d'un collaborateur.
   - Selectionnez son **role** (Commercial, Comptable, RH, etc.).
   - Cliquez sur **Inviter**.
   - Repetez l'operation pour chaque collaborateur.
   - Vous pouvez passer cette etape et inviter plus tard.
5. Cliquez sur **Acceder au tableau de bord** pour terminer l'assistant.

## Modifier les informations plus tard

Toutes les informations saisies peuvent etre modifiees ulterieurement depuis **Administration** > **Parametres entreprise**.

## Etapes suivantes

Apres avoir configure votre entreprise, effectuez les [premiers reglages](/premiers-pas/premiers-reglages) pour preparer OwoDesk a une utilisation quotidienne.
