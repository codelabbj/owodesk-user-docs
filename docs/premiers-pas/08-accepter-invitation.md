# Accepter une invitation

Si un collègue vous a invité dans son entreprise OwoDesk :

1. Connectez-vous (ou créez un compte avec le **même email** que l'invitation).
2. Ouvrez **Invitations** dans le menu.
3. Acceptez ou refusez l'invitation.
4. Vous accédez ensuite à l'entreprise avec le rôle qui vous a été attribué.

:::tip
Vous pouvez appartenir à plusieurs entreprises et basculer entre elles depuis le menu profil.
:::
