# Mode hors ligne

OwoDesk peut continuer à fonctionner avec une connexion limitée ou coupée.

## Ce qui reste disponible

- Consultation des données déjà chargées
- Certaines actions en file d'attente (synchronisation à la reconnexion)

## Ce qui nécessite Internet

- Envoi de factures certifiées
- Paiements PAL
- Synchronisation temps réel avec l'équipe
- Assistant Owo (IA)

## Indicateur

Le tableau de bord affiche un bandeau lorsque l'application est en **mode dégradé**.
