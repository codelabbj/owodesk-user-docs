# Premiers réglages recommandés

Après l'[onboarding](/premiers-pas/configurer-entreprise), le **propriétaire** doit effectuer ces réglages pour que l'équipe puisse travailler sereinement.

## Ce que le propriétaire doit faire

1. **Souscrire à un abonnement** — **Abonnements** : choisir un forfait et payer (sinon les modules restent verrouillés).

2. **Vérifier les informations entreprise** — **Paramètres** → **Entreprise** : nom, forme juridique, RCCM, pays, adresse, téléphone, identifiant fiscal (IFU, NIF, NINEA), logo et signature.

3. **Configurer la facturation** — **Paramètres** → **Facturation** : mode de facturation (simple, normalisée ou les deux), règles de TVA sur les lignes, message de pied de page.

4. **Activer les modules nécessaires** — **Abonnements** : CRM, Facturation, Stock, Comptabilité, RH, POS, E-commerce… selon votre activité.

5. **Préparer le stock** *(si vous vendez des produits)* — **Stock** → **Entrepôts**, **Produits**, **Niveaux de stock** : au minimum un entrepôt, vos articles et les quantités de départ.

6. **Inviter les collaborateurs** — **Paramètres** → **Collaborateurs** : envoyer les invitations avec le bon rôle (Commercial, Comptable, RH…).

7. **Vérifier les rôles et permissions** — **Paramètres** → **Rôles & permissions** : chaque personne ne voit que ce dont elle a besoin.

8. **Configurer les paiements PAL** *(optionnel)* — **Paramètres** → **Paiements** : identifiants PAL et activation par module (facturation, paie, POS…).

9. **Configurer la paie** *(si module RH)* — **Paramètres** → **Paie** : taux CNSS et options de bulletin avant la première paie.

10. **Tester la certification fiscale** *(Bénin, factures normalisées)* — connecter le terminal MCF, vérifier l'IFU, tester depuis **Facturation** → **Fiscal / e-facture**.

## Rappel

Le tableau de bord peut paraître vide au début : les indicateurs apparaissent après vos premières factures, mouvements de stock ou opportunités commerciales.
