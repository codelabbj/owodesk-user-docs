# Comprendre les modules

OwoDesk est organise en **modules**. Chaque module correspond a un domaine de gestion de votre entreprise.

## Principe

1. OwoDesk fonctionne par modules independants mais interconnectes.
2. Chaque module couvre un metier specifique : CRM, facturation, stock, comptabilite, RH, etc.
3. Les modules accessibles dependent de votre **forfait d'abonnement**.
4. Les modules actifs apparaissent dans le **menu lateral** de l'application.
5. Le tableau de bord affiche les indicateurs des modules actifs.

## Modules disponibles

| Module | Description |
|--------|-------------|
| **Tableau de bord** | Indicateurs cles, tendances, alertes et activite recente |
| **CRM** | Gestion des contacts, entreprises, opportunites et pipeline de vente |
| **Facturation** | Factures, devis, encaissements et certification fiscale |
| **Stock** | Catalogue produits, entrepots, mouvements, inventaires et achats fournisseurs |
| **Comptabilite** | Plan comptable, ecritures, tresorerie, revenus, depenses et rapports financiers |
| **RH** | Fiches employes, paie, conges, pointage, demandes de documents |
| **Point de vente (POS)** | Ventes au comptoir, sessions de caisse, tickets certifies |
| **E-Commerce** | Boutique en ligne, publication de produits, commandes web |
| **Projets** | Projets, taches et collaboration d'equipe |
| **Demandes & approbations** | Decaissements, fournitures, RH, validation avec circuit d'approbation |
| **Assistant Owo** | Assistant IA pour repondre a vos questions sur l'utilisation (icone en bas a droite) |

## Comment acceder a un module ?

1. Les modules actifs sont listes dans le **menu lateral** a gauche.
2. Le proprietaire du compte peut consulter le detail de son forfait dans **Abonnements** (menu lateral).
3. L'onglet **Modules Explorer** indique quels modules sont **inclus** ou **verrouilles** selon votre plan.
4. Pour debloquer un module verrouille, changez de forfait depuis la page **Abonnements**.

## Module verrouille ?

Si un module affiche un cadenas dans le menu lateral :

1. **Abonnement insuffisant** : le module n'est pas inclus dans votre plan. Le proprietaire peut changer de forfait depuis **Abonnements**.
2. **Permissions insuffisantes** : votre role ne vous donne pas acces a ce module. Le proprietaire peut ajuster les droits dans **Parametres** > **Roles & permissions**.
