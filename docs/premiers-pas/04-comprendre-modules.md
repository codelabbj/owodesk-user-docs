# Comprendre les modules

OwoDesk est organise en **modules**. Chaque module correspond a un domaine de gestion de votre entreprise.

## Principe

1. OwoDesk fonctionne par modules independants mais interconnectes.
2. Chaque module couvre un metier specifique : CRM, vente, stock, comptabilite, RH, etc.
3. Vous activez uniquement les modules dont vous avez besoin.
4. Les modules actifs apparaissent dans le menu lateral de l'application.
5. Le tableau de bord affiche les indicateurs des modules actifs.

## Modules disponibles

| Module | Description |
|--------|-------------|
| **Tableau de bord** | Indicateurs cles, tendances, alertes et activite recente |
| **CRM** | Gestion des contacts, entreprises, opportunites et pipeline de vente |
| **Facturation** | Factures, devis, bons de livraison, encaissements et certification fiscale |
| **Stock** | Catalogue produits, entrepots, mouvements, inventaires et commandes fournisseurs |
| **Ventes** | Devis, commandes clients, bons de livraison |
| **Achats** | Commandes fournisseurs, receptions, suivi des achats |
| **Comptabilite** | Plan comptable, ecritures, tresorerie, rapports financiers |
| **Depenses** | Enregistrement et suivi des depenses |
| **RH** | Fiches employes, paie, conges, pointage, demandes de documents |
| **Point de vente (POS)** | Ventes au comptoir, sessions de caisse, tickets certifies |
| **E-Commerce** | Boutique en ligne, publication de produits, commandes web |
| **Projets** | Projets, taches, echeances, collaboration d'equipe |
| **Demandes internes** | Decaissements, fournitures, RH, validation avec circuit d'approbation |
| **Support** | Tickets d'assistance client |
| **Marketing** | Campagnes marketing et gestion des contacts |
| **WhatsApp** | Messagerie WhatsApp Business integree |
| **Assistant Owo** | Assistant IA pour repondre a vos questions sur l'utilisation |

## Comment activer un module ?

1. Allez dans **Administration** > **Abonnement et modules**.
2. Consultez la liste des modules inclus dans votre plan.
3. Activez ou desactivez les modules selon vos besoins.
4. Les modifications sont prises en compte immediatement.

## Module verrouille ?

Si un module affiche un cadenas dans le menu lateral :

1. **Abonnement insuffisant** : le module n'est pas inclus dans votre plan. Contactez le proprietaire pour changer de forfait.
2. **Permissions insuffisantes** : votre role ne vous donne pas acces a ce module. Le proprietaire peut ajuster les droits dans **Administration** > **Collaborateurs et roles**.
