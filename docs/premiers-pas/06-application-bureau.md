# Application bureau (ordinateur)

OwoDesk est disponible en **application de bureau** (Windows, macOS, Linux) en plus du navigateur web.

## Avantages

- Icône sur le bureau, lancement rapide
- Fonctionnement hors ligne partiel (lecture des données en cache)
- Adapté aux points de vente et bureaux avec connexion instable

## Installation

:::info À compléter
Ajouter le lien de téléchargement et les captures d'installation par système.
:::

1. Téléchargez l'installateur depuis le site OwoDesk.
2. Installez et ouvrez l'application.
3. Connectez-vous avec les mêmes identifiants que sur le web.

## Bon à savoir

Les données sensibles restent synchronisées avec le serveur dès que la connexion est rétablie.
