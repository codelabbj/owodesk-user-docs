# Application de bureau

OwoDesk est disponible en **application de bureau** pour Windows, macOS et Linux. Cette application offre les memes fonctionnalites que la version web, avec des avantages supplementaires.

## Avantages de l'application de bureau

- Lancement rapide depuis une icone sur le bureau.
- Fonctionnement hors ligne : consultation des donnees mises en cache.
- Synchronisation automatique a la reconnexion.
- Ideal pour les points de vente et les bureaux avec connexion instable.

## Installation

1. Rendez-vous sur la page **Telechargements** depuis le menu de l'application ou sur [owo.bj](https://owo.bj).
2. Telechargez l'installateur correspondant a votre systeme d'exploitation :
   - **Windows** : fichier `.exe` ou `.msi`.
   - **macOS** : fichier `.dmg`.
   - **Linux** : fichier `.AppImage` ou `.deb`.
3. Ouvrez le fichier telecharge et suivez les instructions d'installation.
4. Lancez l'application apres l'installation.
5. Connectez-vous avec votre **email** et **mot de passe** OwoDesk (les memes identifiants que sur le web).

## Utilisation hors ligne

L'application de bureau conserve en cache les donnees recentes. En cas de coupure Internet :

1. Les donnees deja chargees restent consultables (contacts, produits, factures).
2. Les actions effectuees hors ligne sont mises en file d'attente.
3. Des que la connexion est retablie, les donnees sont synchronisees automatiquement avec le serveur.

## Desinstallation

1. **Windows** : ouvrez le Panneau de configuration > Programmes > Desinstaller un programme. Selectionnez OwoDesk et cliquez sur Desinstaller.
2. **macOS** : faites glisser l'application OwoDesk du dossier Applications vers la Corbeille.
3. **Linux** : utilisez le gestionnaire de paquets de votre distribution.
