# Categories de depenses

Les categories permettent de classer les depenses pour une analyse financiere par poste de charge.

## Categories par defaut

A l'installation, les categories suivantes sont creees automatiquement :

- Loyer
- Electricite
- Eau
- Telephone et Internet
- Transport et carburant
- Fournitures de bureau
- Assurance
- Maintenance et reparations
- Marketing et publicite
- Frais bancaires
- Salaires
- Impots et taxes

## Creer une categorie personnalisee

1. Ouvrez **Depenses** -> **Categories**.
2. Cliquez **Nouvelle categorie**.
3. Saisissez les informations :
   - **Nom** de la categorie (ex : "Formation du personnel").
   - **Couleur** pour l'identifier visuellement dans les graphiques.
   - **Icône** representative.
4. Cliquez **Enregistrer**.

## Modifier une categorie

1. Ouvrez **Depenses** -> **Categories**.
2. Cliquez sur la categorie a modifier.
3. Mettez a jour les champs souhaites.
4. Cliquez **Enregistrer**.

## Supprimer une categorie

1. Ouvrez la categorie a supprimer.
2. Cliquez le menu **Actions** -> **Supprimer**.
3. Confirmez la suppression. Les depenses de cette categorie ne seront plus categorisees.
