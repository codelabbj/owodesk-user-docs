# Creer un contact

Les contacts sont les personnes physiques avec qui vous entretenez une relation commerciale.

## Creer un contact

1. Ouvrez **CRM** > **Contacts**.
2. Cliquez sur **Nouveau contact**.
3. Renseignez les informations :
   - **Civilite** (M., Mme, Mlle)
   - **Nom** et **Prenom**
   - **Email** et **Telephone**
   - **Fonction** et **Service**
4. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/crm/contact-creation.png" alt="Creation d'un contact" />

## Lier un contact a une entreprise

1. Lors de la creation ou de la modification d'un contact, ouvrez la section **Entreprise liee**.
2. Selectionnez une entreprise existante dans la liste.
3. Si l'entreprise n'existe pas encore, creez-la d'abord depuis **CRM** > **Entreprises**.
4. Enregistrez le contact.

## Ajouter une source

1. Dans la fiche contact, ouvrez le champ **Source**.
2. Choisissez parmi les options :
   - **Site web** (formulaire de contact)
   - **Recommandation** (bouche-a-oreille)
   - **Reseau social** (LinkedIn, Facebook)
   - **Evenement** (salon, conference)
   - **Appel entrant**
   - **Autre**
3. Enregistrez le contact.

## Scoring et tags

1. Dans la fiche contact, le **score** est un nombre de 0 a 100 qui represente la qualite du lead.
2. Modifiez le score manuellement ou laissez le systeme le calculer selon les interactions.
3. Ajoutez des **tags** en saisissant un mot-cle dans le champ prevu et en validant avec **Entree**.
4. Les tags permettent de filtrer et de segmenter vos contacts.

## Filtrer les contacts

1. Ouvrez **CRM** > **Contacts**.
2. Utilisez les filtres en haut de la liste :
   - **Source** : afficher les contacts d'une source specifique
   - **Entreprise** : filtrer par entreprise liee
   - **Tags** : saisir un tag pour afficher les contacts correspondants
   - **Date de creation** : periode de creation
3. Les resultats se mettent a jour automatiquement.
