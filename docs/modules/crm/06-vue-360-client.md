# Vue 360 client

La vue 360 client regroupe toutes les informations liees a une entreprise sur une seule page.

## Acceder a la vue 360 d'une entreprise

1. Ouvrez **CRM** > **Entreprises**.
2. Cliquez sur le nom d'une entreprise dans la liste.
3. La **vue 360** s'affiche automatiquement avec tous les onglets.

<DocImage src="/img/guides/crm/vue-360.png" alt="Vue 360 client" />

## Voir les contacts lies

1. Dans la vue 360, cliquez sur l'onglet **Contacts**.
2. La liste des contacts rattaches a l'entreprise s'affiche.
3. Cliquez sur un contact pour ouvrir sa fiche detaillee.
4. Depuis cet onglet, vous pouvez ajouter un nouveau contact lie.

## Voir les affaires en cours et leur valeur totale

1. Dans la vue 360, cliquez sur l'onglet **Affaires**.
2. Consultez la liste des affaires en cours avec leur montant et leur probabilite.
3. Le **montant total** des affaires en cours est affiche en haut de l'onglet.
4. Cliquez sur une affaire pour acceder a ses details.

## Voir les factures, devis, commandes et projets

1. Dans la vue 360, utilisez les onglets correspondants :
   - **Factures** : historique des factures emises
   - **Devis** : propositions commerciales
   - **Commandes** : commandes clients
   - **Projets** : projets lies a l'entreprise
2. Chaque onglet affiche le montant total et le statut des documents.
3. Cliquez sur un document pour l'ouvrir.

## Activites recentes

1. Dans la vue 360, l'onglet **Activites** affiche le fil chronologique.
2. Consultez les dernieres interactions : appels, emails, reunions, notes.
3. La date et le type d'activite sont affiches pour chaque entree.
4. Cliquez sur une activite pour voir le detail ou ajouter un commentaire.
