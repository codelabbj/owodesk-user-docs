# Pipelines de vente

Un pipeline de vente modelise les etapes de votre processus commercial, de la prospection a la conclusion.

## Creer un pipeline de vente

1. Ouvrez **CRM** > **Pipelines**.
2. Cliquez sur **Nouveau pipeline**.
3. Saisissez un **nom** (ex. "Pipeline standard", "Pipeline express").
4. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/crm/pipeline-creation.png" alt="Creation d'un pipeline" />

## Definir les etapes

1. Depuis la fiche du pipeline, cliquez sur **Ajouter une etape**.
2. Saisissez le **nom** de l'etape :
   - **Lead** : premier contact
   - **Qualification** : besoin identifie
   - **Proposition** : devis ou proposition envoyee
   - **Negociation** : echange sur les conditions
   - **Gagne** / **Perdu** : conclusion
3. Definissez une **probabilite par defaut** pour chaque etape (ex. Lead = 10%, Proposition = 50%, Negociation = 75%).
4. Ajoutez autant d'etapes que necessaire.

## Ordonner les etapes

1. Dans la fiche pipeline, les etapes apparaissent dans l'ordre de definition.
2. Pour reordonner, utilisez les icones de **glisser-deposer** (six points) a gauche du nom.
3. La premiere etape est le debut du pipeline, la derniere est la conclusion.

## Configurer les probabilites par defaut

1. Pour chaque etape, saisissez une **probabilite** (en pourcentage) dans le champ dedie.
2. Cette probabilite est appliquee automatiquement aux nouvelles affaires placees dans cette etape.
3. Vous pouvez la modifier individuellement sur chaque affaire.
