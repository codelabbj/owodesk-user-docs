# Vue 360°

Onglet **Vue 360°** (`/app/crm/customer-360`). Synthese commerciale par entreprise.

## Selection

Liste deroulante **entreprise**. Sans selection : message **Selectionnez une entreprise**.

## Identite entreprise

Avatar, nom, badges secteur et taille, lien site web.

## Indicateurs (6 cartes)

| Indicateur | Signification |
|------------|---------------|
| **Valeur pipeline** | Montant des affaires ouvertes |
| **Total facture** | Montant facture |
| **Contacts** | Nombre de contacts lies |
| **Devis** | Nombre de devis |
| **Commandes** | Nombre de commandes |
| **Factures** | Nombre de factures |

## Sections (page unique, sans sous-onglets)

### Affaires

Liste : titre, valeur, badge etape, probabilite.

### Projets

Liste : nom, badge statut.

### Activites recentes

Liste : type (emoji), description, date.

Vue **lecture seule** alimentee par l'API customer 360. Pour modifier les affaires ou activites, utilisez [Accueil](/modules/crm/accueil) ou la [fiche contact](/modules/crm/contacts).

## Chargement

Spinner pendant le chargement des donnees.
