# Gerer les affaires (deals)

Une affaire (ou deal) represente une opportunite commerciale en cours dans votre pipeline.

## Creer une affaire

1. Ouvrez **CRM** > **Pipeline**.
2. Cliquez sur **Nouvelle affaire** (ou sur le bouton "+" dans une colonne).
3. Renseignez les informations :
   - **Titre** de l'affaire (ex. "Vente de solutions logicielles")
   - **Montant** estime
   - **Probabilite** de conclusion (en %)
   - **Etape** dans le pipeline
4. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/crm/affaire-creation.png" alt="Creation d'une affaire" />

## Lier une affaire a un contact

1. Dans la fiche de creation ou de modification, ouvrez la section **Contact**.
2. Selectionnez un **contact** existant.
3. Vous pouvez aussi lier une **entreprise** en plus du contact.
4. Enregistrez l'affaire.

## Assigner un responsable

1. Dans la fiche affaire, ouvrez le champ **Responsable**.
2. Selectionnez un collaborateur de votre organisation.
3. Le responsable recevra les notifications liees a cette affaire.

## Definir une date de cloture previsionnelle

1. Dans la fiche affaire, ouvrez le champ **Date de cloture prevue**.
2. Selectionnez une date dans le calendrier.
3. Cette date sert de reference pour le suivi des echances.

## Filtrer les affaires

1. Ouvrez **CRM** > **Pipeline**.
2. Utilisez les filtres disponibles :
   - **Par etape** : visualiser une colonne specifique
   - **Par contact** : afficher les affaires d'un contact
   - **Par responsable** : filtrer par commercial
   - **Par montant** : seuil minimal ou maximal
3. Le pipeline se met a jour dynamiquement.
