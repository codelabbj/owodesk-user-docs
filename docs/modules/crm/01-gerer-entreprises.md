# Gerer les entreprises

Les fiches entreprises regroupent les informations legales et commerciales de vos clients et prospects.

## Creer une entreprise

1. Ouvrez **CRM** > **Entreprises**.
2. Cliquez sur **Nouvelle entreprise**.
3. Renseignez les champs obligatoires :
   - **Nom** de l'entreprise
   - **Secteur d'activite** (liste predefinie)
   - **Taille** (effectif, chiffre d'affaires)
4. Remplissez les informations complementaires :
   - **Site web** et **adresse email**
   - **Numero de telephone** et **adresse postale**
   - **Pays** et **ville**
5. Cliquez sur **Enregistrer**.

## Modifier une entreprise

1. Ouvrez **CRM** > **Entreprises**.
2. Cliquez sur le nom de l'entreprise dans la liste.
3. Modifiez les champs souhaites (secteur, taille, coordonnees).
4. Cliquez sur **Enregistrer** pour confirmer les modifications.

## Supprimer une entreprise

1. Ouvrez **CRM** > **Entreprises**.
2. Cliquez sur le nom de l'entreprise a supprimer.
3. Dans la fiche, cliquez sur **Supprimer** (icone poubelle).
4. Confirmez la suppression dans la boite de dialogue.

> La suppression est irreversible. Les contacts lies a l'entreprise seront conserves mais detaches.

## Vue 360 client

1. Depuis la liste des entreprises, cliquez sur une entreprise.
2. La **vue 360** affiche en un coup d'oeil :
   - Les **contacts** rattaches
   - Les **affaires en cours** et leur montant total
   - Les **activites recentes** (appels, emails, reunions)
   - Les **factures, devis, commandes et projets** lies
3. Utilisez les onglets pour naviguer entre les sections.
