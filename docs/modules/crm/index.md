# CRM

Le module CRM (Customer Relationship Management) vous permet de gerer vos relations commerciales : contacts, entreprises, pipelines de vente, affaires et activites.

## Fonctionnalites

- **Entreprises** — Fiches completes avec secteur, taille, site web, adresse
- **Contacts** — Gestion des prospects et clients avec scoring et tags
- **Pipelines de vente** — Personnalisables par etapes
- **Affaires (Deals)** — Opportunites commerciales avec suivi de probabilite
- **Activites** — Appels, emails, reunions, notes lies aux contacts/affaires
- **Vue 360 client** — Toutes les informations clients en un coup d'oeil
- **Rappels d'activites** — Notifications automatiques pour les activites planifiees
