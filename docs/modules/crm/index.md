# CRM

Gérez vos relations clients : contacts, entreprises, opportunités et activités.

## Sections principales

| Section | Usage |
|---------|-------|
| **Pipeline** | Suivre les deals par étape (kanban) |
| **Contacts** | Personnes avec qui vous travaillez |
| **Entreprises** | Sociétés clientes ou prospects |
| **Activités** | Appels, rendez-vous, tâches de suivi |

## Exemple concret

Un prospect appelle → créez un **contact** et une **opportunité** → déplacez-la dans le pipeline jusqu'à la signature.

## Parcours liés

Pour un parcours commercial complet jusqu'à la facture, consultez [De la vente à la facture](/modules/facturation/parcours/vente-a-facture) (module Facturation).
