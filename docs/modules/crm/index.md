# CRM

Le module **CRM** (`/app/crm`) centralise entreprises, contacts, pipelines et affaires commerciales.

| Onglet dans l'app | Documentation |
|-------------------|---------------|
| **Accueil** | [Accueil (pipeline)](/modules/crm/accueil) |
| **Entreprises** | [Entreprises](/modules/crm/entreprises) |
| **Contacts** | [Contacts](/modules/crm/contacts) |
| **Pipelines** | [Pipelines](/modules/crm/pipelines) |
| **Vue 360°** | [Vue 360°](/modules/crm/vue-360) |

## Parcours type

1. Configurez vos [pipelines](/modules/crm/pipelines) et etapes.
2. Creez [entreprises](/modules/crm/entreprises) et [contacts](/modules/crm/contacts).
3. Gerez les affaires sur l'onglet [Accueil](/modules/crm/accueil) (Kanban).
4. Consultez la synthese client dans [Vue 360°](/modules/crm/vue-360).

## Centre d'aide integre

Le bouton **Centre d'aide** decrit chaque onglet du module.

## Pages hors menu

Les routes `/app/crm/dashboard`, `/app/crm/deals` et `/app/crm/activities` existent dans l'application mais ne figurent pas dans les onglets CRM. Les affaires et activites sont gerees depuis l'onglet **Accueil** et les fiches contact.
