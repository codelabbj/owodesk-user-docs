# De la vente à la facture

Parcours complet : du premier contact client jusqu'à la facture payée.

## Étapes

**Contact CRM** → **Opportunité** → **Devis** → **Facture** → **Encaissement** → **Comptabilité**

1. **CRM** — Créer le contact et l'entreprise cliente
2. **Pipeline** — Ouvrir une opportunité et avancer les étapes
3. **Facture Pro** — Émettre un devis
4. **Facturation** — Convertir le devis en facture
5. **Stock** — Vérifier / déstocker si produits physiques
6. **Comptabilité** — Enregistrer la recette
7. **(Bénin)** — Certifier la facture via e-MECeF si requis
