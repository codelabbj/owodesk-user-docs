# Creer un devis

Un devis est une proposition commerciale qui peut etre convertie en facture apres acceptation.

## Creer un devis

1. Ouvrez **Facturation** > **Devis** (ou **Factures** > **Nouvelle facture**).
2. Dans le champ **Type de document**, selectionnez **Devis**.
3. Selectionnez le **client**.
4. Ajoutez les **lignes d'articles** avec quantite et prix unitaire.
5. Configurez les taxes et les remises si necessaire.
6. Cliquez sur **Enregistrer** pour creer le devis en brouillon.

<DocImage src="/img/guides/facturation/devis-creation.png" alt="Creation d'un devis" />

## Finaliser un devis

1. Ouvrez le devis en statut **Brouillon**.
2. Verifiez l'ensemble des informations.
3. Cliquez sur **Finaliser**.
4. Le devis passe en statut **Envoye** (ou **Finalise** selon la configuration).
5. Le PDF du devis est genere automatiquement.

## Envoyer un devis au client

1. Depuis la fiche du devis finalise, cliquez sur **Envoyer par email**.
2. L'adresse email du client est pre-remplie.
3. Modifiez l'objet et le message si necessaire.
4. Cliquez sur **Envoyer**.
5. Un email avec le PDF en piece jointe est expedie au client.

## Accepter un devis

1. Depuis la fiche du devis, cliquez sur **Accepter**.
2. Le statut passe a **Accepte**.
3. Vous pouvez noter les conditions d'acceptation (date, commentaires).
4. Un devis accepte peut etre converti en facture.

## Convertir un devis accepte en facture

1. Depuis la fiche du devis (statut **Accepte**), cliquez sur **Convertir en facture**.
2. Une nouvelle facture est creee avec les memes informations :
   - Client, lignes d'articles, quantites, prix, taxes
3. Modifiez les elements si necessaire (date, conditions de paiement).
4. Enregistrez la facture en brouillon ou finalisez-la directement.

## Creer un devis depuis une facture brouillon

1. Ouvrez une facture en statut **Brouillon**.
2. Cliquez sur **Dupliquer en devis**.
3. Un nouveau devis est cree avec les memes donnees.
4. Le type de document est automatiquement bascule en **Devis**.
5. Finalisez et envoyez le devis au client.
