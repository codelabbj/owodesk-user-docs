# Poser une question a l'Assistant Owo

L'Assistant Owo est une intelligence artificielle capable de repondre a vos questions en utilisant les donnees de votre organisation.

## Ouvrir l'assistant Owo depuis l'application

1. Dans l'application, cliquez sur le bouton **Owo** situe en bas a droite de l'ecran.
2. Un panneau de discussion s'ouvre sur le cote droit.
3. La zone de saisie est prete a recevoir votre question.

<DocImage src="/img/guides/assistant-owo/panneau-owo.png" alt="Panneau de discussion de l'assistant Owo" />

## Poser une question en langage courant

1. Dans le champ de saisie, tapez votre question comme vous le feriez avec un collaborateur.
2. Exemples de questions possibles :
   - "Combien de contacts avons-nous dans le CRM ?"
   - "Quelles sont les factures impayees ce mois-ci ?"
   - "Montre-moi les produits en alerte de stock"
   - "Quel est le chiffre d'affaires du trimestre ?"
3. Appuyez sur **Entree** ou cliquez sur le bouton **Envoyer**.
4. Owo analyse votre question et consulte les donnees de votre organisation.

## Obtenir une reponse avec contexte (donnees de l'organisation)

1. Owo formule une reponse en francais en tenant compte de vos donnees reelles.
2. La reponse peut inclure :
   - Des **chiffres** et **statistiques** tires de votre base
   - Des **liens** vers les ecrans concernes (factures, contacts, produits)
   - Des **suggestions d'actions** complementaires
3. La reponse s'affiche dans le panneau de discussion.

<DocImage src="/img/guides/assistant-owo/reponse-owo.png" alt="Reponse de l'assistant Owo avec donnees contextuelles" />

## Conversation avec historique (plusieurs messages)

1. Owo conserve l'historique de la conversation en cours.
2. Vous pouvez poser des questions de suivi sans rappeler le contexte :
   - "Et pour le mois dernier ?" (Owo comprend qu'il s'agit toujours des factures)
   - "Peux-tu detailler ?" (Owo approfondit le dernier sujet aborde)
3. Chaque echange s'affiche dans le panneau sous forme de bulles de conversation.
4. Pour demarrer une nouvelle conversation, cliquez sur **Nouvelle discussion**.
5. L'historique des discussions precedentes est accessible depuis l'icone **Historique**.

<DocImage src="/img/guides/assistant-owo/conversation-owo.png" alt="Conversation avec historique dans l'assistant Owo" />

## Streaming : reponse en temps reel

1. Apres avoir pose votre question, la reponse commence a s'afficher immediatement.
2. Le texte apparait **lettre par lettre** (effet de streaming).
3. Vous pouvez lire le debut de la reponse pendant que la suite se construit.
4. L'affichage en streaming offre une experience interactive sans temps d'attente.
5. Vous pouvez interrompre la generation a tout moment en cliquant sur **Arreter**.

## Assistant public (widget sur le site)

1. L'assistant Owo peut etre integre a votre site web via un widget.
2. Les visiteurs du site peuvent poser des questions sans authentification.
3. Le widget public repond uniquement aux informations non sensibles (documentation, informations generales).
4. Pour installer le widget sur votre site, contactez votre administrateur technique.
5. Les conversations du widget public sont independantes de l'historique de l'application.
