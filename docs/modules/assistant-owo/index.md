# Assistant Owo

L'Assistant Owo est une intelligence artificielle integree a l'application qui repond a vos questions en langage courant en utilisant les donnees de votre organisation.

## Fonctionnalites

- **Questions en langage naturel** -- Posez vos questions comme a un collaborateur
- **Contexte organisationnel** -- Les reponses tiennent compte de vos donnees (clients, factures, stocks)
- **Conversation** -- Historique conserve pour un suivi continu
- **Streaming** -- Reponse affichee en temps reel, lettre par lettre
- **Widget public** -- Assistant accessible depuis le site web sans authentification
