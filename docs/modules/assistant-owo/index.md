# Assistant Owo

L'assistant intelligent intégré à OwoDesk répond à vos questions en langage naturel.

## Comment l'utiliser

1. Cliquez sur le bouton **Owo** (en bas à droite de l'écran).
2. Posez votre question : *« Comment créer une facture ? »*, *« Où voir le stock ? »*
3. L'assistant vous guide avec des réponses contextualisées à votre entreprise.

## Limites

- Nécessite une connexion Internet
- Ne remplace pas la validation comptable ou juridique humaine
- Les actions sensibles (paiement, suppression) restent manuelles
