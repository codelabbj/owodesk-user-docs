# Confirmer une commande

1. Ouvrez **Achats** -> **Commandes fournisseur**.
2. Trouvez la commande au statut "brouillon".
3. Ouvrez la commande.
4. Cliquez **Confirmer**.
5. La commande passe au statut **"confirme"**. Elle n'est plus modifiable directement mais peut etre receptionnee.

## Statuts d'une commande fournisseur

| Statut | Description |
|---|---|
| Brouillon | En cours de saisie, modifiable |
| Confirme | Validee et envoyee au fournisseur |
| Partiellement recu | Au moins une ligne receptionnee |
| Recu | Toutes les lignes receptionnees |
| Paye | La commande a ete payee |

## Actions possibles

1. Depuis le statut "confirme", vous pouvez :
   - **Receptionner** les marchandises (voir [Receptionner des marchandises](/modules/achats/receptionner-marchandises)).
   - **Imprimer** le bon de commande.
   - **Annuler** la commande si necessaire.

2. Une commande annulee ne peut plus etre receptionnee.
