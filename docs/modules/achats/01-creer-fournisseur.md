# Creer un fournisseur

1. Ouvrez **Achats** -> **Fournisseurs**.
2. Cliquez **Nouveau fournisseur**.
3. Remplissez les informations obligatoires :
   - **Nom** du fournisseur (ex : "Distribtech SARL").
   - **IFU** (Identifiant Fiscal Unique) si applicable.
4. Renseignez les coordonnees :
   - **Email** et **telephone**.
   - **Adresse** complete.
5. Optionnellement, liez a une **entreprise** existante dans le CRM en la selectionnant dans le champ dedie.
6. Cliquez **Enregistrer** pour creer le fournisseur.

## Modifier un fournisseur

1. Ouvrez **Achats** -> **Fournisseurs**.
2. Cliquez sur le nom du fournisseur dans la liste.
3. Modifiez les champs souhaites.
4. Cliquez **Enregistrer** pour appliquer les modifications.

## Supprimer un fournisseur

1. Ouvrez la fiche du fournisseur.
2. Cliquez le menu **Actions** -> **Supprimer**.
3. Confirmez la suppression. Le fournisseur est supprime definitivement.
