# Creer une commande fournisseur

1. Ouvrez **Achats** -> **Commandes fournisseur**.
2. Cliquez **Nouveau bon de commande**.
3. Selectionnez le **fournisseur** dans la liste des fournisseurs enregistres.
4. Une reference automatique est attribuee (format PO-XXXXX).
5. Ajoutez des lignes de commande :
   - Cliquez **Ajouter une ligne**.
   - Selectionnez un **produit** dans le catalogue.
   - Indiquez la **quantite** commandee.
   - Saisissez le **cout unitaire** (prix d'achat convenu avec le fournisseur).
   - Le montant total de la ligne est calcule automatiquement.
6. Verifiez les totaux :
   - **Sous-total** (somme des lignes).
   - **Taxes** appliquees selon la configuration.
   - **Total TTC**.
7. Cliquez **Enregistrer** pour sauvegarder en brouillon.

La commande est enregistree au statut "brouillon". Vous pouvez la modifier ou la supprimer tant qu'elle n'est pas confirmee.
