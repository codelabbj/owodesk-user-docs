# Gerer les entrepots

Les entrepots representent les lieux physiques de stockage de vos produits.

## Creer un entrepot

1. Ouvrez **Stock** > **Entrepots**.
2. Cliquez sur **Nouvel entrepot**.
3. Saisissez le **nom** de l'entrepot (ex. "Entrepot principal", "Boutique Cotonou").
4. Renseignez l'**adresse** complete.
5. Designez un **responsable** (optionnel).
6. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/stock/entrepot-creation.png" alt="Creation d'un entrepot" />

## Configurer les coordonnees GPS (latitude/longitude)

1. Dans la fiche de l'entrepot, ouvrez la section **Coordonnees GPS**.
2. Saisissez la **latitude** (ex. 6.3570).
3. Saisissez la **longitude** (ex. 2.4290).
4. Les coordonnees sont utilisees pour le routage e-commerce et les cartes.
5. Vous pouvez aussi cliquer sur la carte interactive pour definir la position.

## Activer la vente depuis l'entrepot

1. Dans la fiche de l'entrepot, activez l'option **Vente autorisee depuis cet entrepot**.
2. Lorsque l'option est active, l'entrepot apparait dans la liste de selection lors de la facturation.
3. Les entrepots sans cette option ne peuvent pas etre selectionnes pour une vente.

## Definir un entrepot comme depot central

1. Dans la fiche de l'entrepot, activez **Depot central**.
2. Le depot central est l'entrepot par defaut lors de la creation de nouveaux produits.
3. Un seul entrepot peut etre le depot central.
4. Les transferts de stock peuvent etre organises depuis ou vers le depot central.

## Configurer le numero WhatsApp

1. Dans la fiche de l'entrepot, renseignez le champ **WhatsApp**.
2. Ce numero est utilise pour les notifications de stock et les confirmations de commandes.
3. Le format international est recommande (ex. +22990123456).
