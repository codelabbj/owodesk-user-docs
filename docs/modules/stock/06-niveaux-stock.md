# Consulter les niveaux de stock

Les niveaux de stock affichent les quantites disponibles par produit et par entrepot.

## Consulter les niveaux de stock

1. Ouvrez **Stock** > **Niveaux de stock**.
2. La liste affiche tous les produits avec leur quantite par entrepot.
3. Les colonnes principales sont :
   - **Produit** : nom et SKU
   - **Entrepot** : lieu de stockage
   - **Quantite** : nombre d'unites disponibles
   - **Seuil d'alerte** : niveau minimum defini
4. Utilisez la barre de recherche pour trouver un produit specifique.

<DocImage src="/img/guides/stock/niveaux-stock.png" alt="Niveaux de stock par entrepot" />

## Filtrer par entrepot

1. En haut de la page, ouvrez le filtre **Entrepot**.
2. Selectionnez un entrepot specifique.
3. La liste se limite aux produits stockes dans cet entrepot.
4. Pour voir tous les entrepots, selectionnez **Tous**.

## Definir un seuil d'alerte

1. Ouvrez **Stock** > **Niveaux de stock**.
2. Cliquez sur une ligne de stock existante.
3. Dans le champ **Seuil d'alerte**, saisissez la quantite minimale.
4. Enregistrez.
5. Lorsque le stock passe en dessous du seuil, une alerte visuelle apparait (icone orange/rouge).
6. Les produits en dessous du seuil sont listes dans le tableau de bord.

## Verifier la disponibilite d'un produit

1. Ouvrez **Stock** > **Niveaux de stock**.
2. Recherchez le produit par nom ou SKU.
3. Consultez la quantite disponible dans chaque entrepot.
4. La colonne **Disponible** indique si la quantite est suffisante pour les commandes en cours.
