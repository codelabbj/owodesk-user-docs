# Gerer les taxes

Les taxes definissent les regles de calcul applicables aux produits lors de la facturation.

## Creer une taxe

1. Ouvrez **Stock** > **Taxes**.
2. Cliquez sur **Nouvelle taxe**.
3. Saisissez le **nom** de la taxe (ex. "TVA 18%", "AIB 1%").
4. Definissez le **taux** (pourcentage) ou le **montant fixe**.
5. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/stock/taxe-creation.png" alt="Creation d'une taxe" />

## Type : specifique ou touristique

1. Lors de la creation, choisissez le **type** de taxe :
   - **Standard** : taxe generale (ex. TVA)
   - **Specifique** : taxe applicable a certains produits uniquement (ex. droit d'accise)
   - **Touristique** : taux reduit pour le secteur touristique
2. Le type determine les regles d'application sur les produits.

## Mode de calcul : pourcentage ou fixe

1. Dans le champ **Mode de calcul**, choisissez :
   - **Pourcentage** : la taxe est calculee comme un % du montant HT (ex. TVA 18%)
   - **Fixe** : la taxe est un montant fixe par unite vendue (ex. 50 FCFA par article)
2. Pour le mode **Pourcentage**, saisissez le taux (ex. 18).
3. Pour le mode **Fixe**, saisissez le montant par unite.

## Activer/desactiver une taxe

1. Dans la liste des taxes, chaque taxe a un bouton **Actif/Inactif**.
2. Desactiver une taxe ne la supprime pas mais l'empeche d'etre selectionnee sur les nouveaux produits.
3. Les produits deja configures avec cette taxe ne sont pas impactes.
4. Pour reactiver, basculez simplement le bouton.
