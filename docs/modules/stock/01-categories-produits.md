# Gerer les categories de produits

Les categories permettent d'organiser vos produits par famille.

## Creer une categorie

1. Ouvrez **Stock** > **Categories**.
2. Cliquez sur **Nouvelle categorie**.
3. Saisissez le **nom** de la categorie (ex. "Boissons", "Electromenager").
4. Ajoutez une **description** (optionnelle).
5. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/stock/categorie-creation.png" alt="Creation d'une categorie" />

## Creer une sous-categorie

1. Depuis la liste des categories, cliquez sur la categorie parente.
2. Cliquez sur **Ajouter une sous-categorie**.
3. Saisissez le **nom** de la sous-categorie.
4. La categorie parente est automatiquement definie.
5. Cliquez sur **Enregistrer**.

## Voir les produits d'une categorie

1. Ouvrez **Stock** > **Categories**.
2. Cliquez sur une categorie pour l'ouvrir.
3. L'onglet **Produits** affiche tous les articles appartenant a cette categorie.
4. Les sous-categories sont affichees avec leur propre nombre de produits.
5. Cliquez sur un produit pour acceder a sa fiche detaillee.
