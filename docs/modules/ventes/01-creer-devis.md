# Creer un devis

1. Ouvrez **Ventes** -> **Devis**.
2. Cliquez **Nouveau devis**. Une reference automatique est attribuee (format DV-XXXX).
3. Renseignez les informations du client :
   - Selectionnez un **contact** ou une **entreprise** existante depuis le CRM.
   - Ou creez un nouveau contact directement (nom, email, telephone).
4. Ajoutez des lignes de devis :
   - Cliquez **Ajouter une ligne**.
   - Selectionnez un **produit** dans le catalogue.
   - Indiquez la **quantite** et le **prix unitaire** (pre-rempli depuis le produit).
   - Le montant total de la ligne est calcule automatiquement.
5. Verifiez les totaux :
   - Le **sous-total** (somme des lignes hors taxes).
   - Les **taxes** appliquees selon la configuration fiscale.
   - Le **total TTC** calcule automatiquement.
6. Cliquez **Enregistrer** pour sauvegarder en brouillon.
7. Pour finaliser le devis, cliquez **Valider**. Le devis passe au statut "envoye" et devient une version figee.

Le devis peut ensuite etre converti en commande client (voir [Convertir un devis en commande](/modules/ventes/convertir-devis-en-commande)).
