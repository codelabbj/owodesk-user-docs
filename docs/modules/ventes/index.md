# Ventes

Devis, commandes clients et bons de livraison. Ce module couvre le cycle de vente de la prospection a la livraison des marchandises.

Les devis se transforment en commandes, les commandes generees declenchent des bons de livraison, et les commandes livrees alimentent la facturation et le stock.

Choisissez dans le menu **l'action a realiser**.
