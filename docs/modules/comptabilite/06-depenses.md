# Depenses

Enregistrez et suivez les depenses de l'entreprise (achats, charges, frais generaux).

## Enregistrer une depense

1. Ouvrez **Comptabilite** -> **Depenses**.
2. Cliquez **Nouvelle depense**.
3. Remplissez :
   - **Montant**, **description**, **categorie**, **date**.
   - **Methode de paiement** (especes, banque, mobile money, carte).
   - **Reference** (optionnel).
4. Ajoutez des **pieces jointes** (jusqu'a 20 fichiers : PDF, images, documents).
5. Cliquez **Enregistrer**.

## Filtrer les depenses

1. Utilisez le filtre de date pour visualiser les depenses d'une periode.
2. Les KPIs affichent le total et la repartition par categorie.

## Telecharger les justificatifs

1. Cochez les depenses concernees.
2. Cliquez **Telecharger les justificatifs**.

## Ecriture comptable automatique

Chaque depense genere une ecriture comptable :
- Debit du compte de charge correspondant a la categorie (classe 6).
- Credit du compte de tresorerie correspondant.
