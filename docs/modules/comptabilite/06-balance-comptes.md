# Balance des comptes

La balance des comptes recapitule l'ensemble des mouvements (debit, credit, solde) pour chaque compte comptable sur une periode donnee.

## Consulter la balance

1. Ouvrez **Comptabilite** -> **Balance**.
2. La balance affiche la liste de tous les comptes avec :
   - **Code** et **nom** du compte.
   - **Total debit** cumule sur la periode.
   - **Total credit** cumule sur la periode.
   - **Solde** (debit - credit).

## Filtrer par classe comptable

1. Utilisez le filtre **Classe** pour afficher uniquement les comptes d'une classe specifique (1 a 9).
2. Exemple : filtrez sur la classe 6 pour voir uniquement les comptes de charges.

## Grand-livre d'un compte

1. Cliquez sur un compte dans la balance pour ouvrir le **grand-livre**.
2. Le grand-livre affiche le detail de toutes les transactions qui ont impacte ce compte :
   - Date et reference de l'ecriture.
   - Libelle de l'operation.
   - Montant debit et credit.
   - Solde apres chaque mouvement.
3. Vous pouvez filtrer par intervalle de dates.
