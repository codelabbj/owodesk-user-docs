# Saisie d'ecritures

Les ecritures comptables sont generees automatiquement par les modules (facturation, achats, depenses, POS). Vous pouvez aussi les saisir manuellement.

## Creer une ecriture

1. Ouvrez **Comptabilite** -> **Ecritures**.
2. Cliquez **Nouvelle ecriture**.
3. Renseignez :
   - **Journal** (VTE, ACH, BQ, CA, POS, MOB, OD).
   - **Date** et **description** de l'ecriture.
4. Ajoutez des lignes :
   - Cliquez **Ajouter une ligne**.
   - Selectionnez un **compte comptable**.
   - Saisissez le **montant debit** ou **credit**.
   - Le total doit etre equilibre (total debit = total credit).
5. Cliquez **Valider l'ecriture**.

## Filtrer les ecritures

1. Utilisez les onglets **Toutes / Brouillon / Postees** pour filtrer par statut.
2. Cherchez par numero, description ou journal.

## Consulter le detail

1. Cliquez sur une ecriture pour developper et voir les lignes.
2. Chaque ligne affiche le compte, le libelle, le debit et le credit.
