# Saisie d'ecritures

## Creer une ecriture comptable

1. Ouvrez **Comptabilite** -> **Ecritures**.
2. Cliquez **Nouvelle ecriture**.
3. Renseignez les informations generales :
   - **Journal** : selectionnez le journal concerne (VTE, ACH, BQ, CA, POS, MOB, OD).
   - **Date** de l'ecriture.
   - **Description** (libelle de l'operation).
4. Ajoutez les lignes d'ecriture :
   - Cliquez **Ajouter une ligne**.
   - Selectionnez un **compte comptable**.
   - Saisissez le **montant debit** ou le **montant credit** (laissez l'autre a 0).
   - Saisissez le **libelle** de la ligne (ex : "Vente de produits").
5. Verifiez l'equilibre : la somme des debits doit etre egale a la somme des credits.
6. Cliquez **Soumettre** pour poster l'ecriture. Le statut passe de "brouillon" a "poste".

## Filtrer les ecritures

1. Depuis la liste des ecritures, utilisez les filtres :
   - **Journal** : filtrer par type de journal.
   - **Statut** : brouillon, poste, annule.
   - **Date** : intervalle de dates.
   - **Compte** : recherche par compte comptable.
2. Les resultats s'affichent en temps reel.
