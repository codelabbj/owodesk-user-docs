# Revenus

La page Revenus centralise toutes les entrees d'argent provenant des factures et des ventes POS.

## Consulter les revenus

1. Ouvrez **Comptabilite** -> **Revenus**.
2. Les revenus sont regroupes par source avec le montant total.
3. Utilisez le filtre de date pour affiner la periode.

## Detail des revenus

Le tableau affiche pour chaque revenu :
- **Source** (facture ou vente POS).
- **Reference** et **client**.
- **Montant** et **methode de paiement**.

## Telecharger les justificatifs

1. Cochez les revenus souhaites.
2. Cliquez **Telecharger les justificatifs**.
3. Les factures et tickets POS sont regroupes en un seul telechargement.
