# Tresorerie et virements

## Consulter les soldes

1. Ouvrez **Comptabilite** -> **Tresorerie**.
2. Les comptes de tresorerie (caisse, banque, mobile money) sont affiches avec leur solde.
3. Cliquez sur un compte pour voir l'historique des transactions.

## Effectuer un virement interne

1. Depuis la page Tresorerie, utilisez le formulaire **Virement interne**.
2. Renseignez :
   - **Compte source** (debite).
   - **Compte destination** (credite).
   - **Montant** et **date**.
3. Cliquez **Valider le virement**.
4. Une ecriture comptable est automatiquement creee dans le journal OD.
