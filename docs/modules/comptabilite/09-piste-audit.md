# Piste d'audit

La piste d'audit enregistre toutes les actions realisees dans le module comptable.

## Consulter la piste d'audit

1. Ouvrez **Comptabilite** -> **Piste d'audit**.
2. Le flux affiche les actions chronologiquement avec :
   - **Action** (creation, modification, suppression).
   - **Type d'entite** et **identifiant**.
   - **Utilisateur** et **date**.

## Filtrer

1. Utilisez la barre de recherche pour filtrer par action, type d'entite ou email.
