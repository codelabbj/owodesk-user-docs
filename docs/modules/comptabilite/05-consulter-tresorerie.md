# Consulter la tresorerie

## Dashboard comptable

Ouvrez **Comptabilite** -> **Tresorerie** pour acceder au tableau de bord de la tresorerie.

## Soldes de tresorerie

Le dashboard affiche les soldes en temps reel pour chaque type de compte :

- **Caisse** (comptes de classe 57) : especes disponibles.
- **Banque** (comptes de classe 51) : soldes bancaires.
- **Mobile money** (comptes de classe 58) : soldes mobile money.

## Revenus et depenses du mois

1. La section **Revenus du mois** totalise les produits (classe 7) enregistres depuis le debut du mois.
2. La section **Depenses du mois** totalise les charges (classe 6) enregistrees depuis le debut du mois.
3. Le **Benefice du mois** est calcule automatiquement : Revenus - Depenses.

## Graphique cashflow 30 jours

1. Le graphique d'evolution du cashflow affiche les entrees et sorties de fonds sur les 30 derniers jours.
2. La courbe superpose le solde cumule pour visualiser la tendance de la tresorerie.
