# Pointage et presence

Suivez les presences des employes avec le systeme de pointage.

## Creer une entree de presence (pointage)

1. Ouvrez **RH** > **Presences**.
2. Cliquez sur **Nouveau pointage**.
3. Selectionnez l'**employe**.
4. Le systeme enregistre automatiquement :
   - **Clock-in** : heure d'arrivee
   - **Clock-out** : heure de depart
   - **Position GPS** (verification de geolocalisation)
5. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/rh/pointage-clock.png" alt="Pointage clock-in / clock-out" />

## Detection de retard

Le systeme compare automatiquement l'heure de clock-in avec le **planning** de l'employe :
- Si l'arrivee depasse l'heure prevue, le pointage est marque **en retard**
- Le retard est visible dans les statistiques

## Consulter les pointages

1. Ouvrez **RH** > **Presences**.
2. Filtrez par **employe** et **periode** (mois, semaine, dates personnalisees).
3. La liste affiche pour chaque jour :
   - Heure d'arrivee et de depart
   - Statut (present, absent, retard, demi-journee)

<DocImage src="/img/guides/rh/pointages-liste.png" alt="Liste des pointages par employe" />

## Statistiques globales

1. Ouvrez **RH** > **Presences** > **Statistiques**.
2. La vue synthetique affiche pour la periode selectionnee :
   - **Nombre de presents**
   - **Nombre d'absents**
   - **Nombre de retards**
   - **Nombre de demi-journees**
   - **Jours ouvres attendus**

## Resume par employe

Sur une periode donnee, consultez le resume individuel :
- Total des heures travaillees
- Nombre de retards
- Jours de presence effectifs
- Jours ouvres attendus vs realises
