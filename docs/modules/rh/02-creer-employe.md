# Creer un employe

Ajoutez un nouvel employe dans le systeme avec ses informations personnelles et professionnelles.

1. Ouvrez **RH** > **Employes**.
2. Cliquez sur **Nouvel employe**.
3. Remplissez les champs obligatoires :
   - **Nom** et **Prenom**
   - **Email** professionnel
   - **Telephone**
4. Selectionnez le **type d'employe** :
   - **Salarie** : numero automatique EMP-XXX
   - **Stagiaire** : numero automatique STG-XXX
   - **Prestataire** : numero automatique PRS-XXX
5. Renseignez les informations professionnelles :
   - **Departement** (liste issue de la gestion des departements)
   - **Poste** occupe
   - **Superviseur** (employe referent)
   - **Date d'embauche**
6. Configurez la remuneration :
   - **Salaire de base**
   - **Regime de paie** (mensuel)
   - **Mode de paiement** (especes, mobile money, virement, cheque)
7. Activez les options selon votre convention :
   - **Conges payes** (droit aux conges)
   - **Transport** (indemnite ou prise en charge)
   - **Logement** (avantage en nature)
   - **Vehicule** (vehicule de fonction)
8. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/rh/creer-employe.png" alt="Formulaire de creation d'un employe" />

Le numero d'employe est genere automatiquement selon le type :
- EMP-001, EMP-002... pour les salaries
- STG-001, STG-002... pour les stagiaires
- PRS-001, PRS-002... pour les prestataires
