# Gerer les departements

Organisez votre structure d'entreprise en creant des departements.

## Creer un departement

1. Ouvrez **RH** > **Departements**.
2. Cliquez sur **Nouveau departement**.
3. Saisissez le **nom** du departement (ex : Commercial, Comptabilite, Technique).
4. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/rh/creer-departement.png" alt="Formulaire de creation d'un departement" />

## Liste des departements

1. Ouvrez **RH** > **Departements**.
2. La liste affiche tous les departements avec leur nombre d'employes.
3. Cliquez sur un departement pour le modifier ou le consulter.

<DocImage src="/img/guides/rh/liste-departements.png" alt="Liste des departements" />
