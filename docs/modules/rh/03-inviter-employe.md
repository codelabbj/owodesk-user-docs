# Inviter un employe

Envoyez une invitation par email a un employe pour qu'il accede a son espace personnel.

1. Ouvrez **RH** > **Employes**.
2. Cliquez sur l'employe a inviter.
3. Cliquez sur le bouton **Inviter**.
4. Un email est envoye automatiquement contenant :
   - Le **code employe** (identifiant unique)
   - Un **code d'activation** de 6 caracteres (valable 72h)
5. L'employe recoit l'email et peut :
   - Acceder a l'application mobile
   - Se connecter avec son **code employe** + **code d'activation**
   - Configurer son **mot de passe** personnel

<DocImage src="/img/guides/rh/inviter-employe.png" alt="Invitation d'un employe" />

## Poste de travail

Apres activation, l'employe peut configurer son poste de travail :
- **Parametres du compte** (mot de passe, profil)
- **Notifications** (email, mobile)
- **Informations personnelles** (coordonnees, urgence)

<DocImage src="/img/guides/rh/poste-travail.png" alt="Configuration du poste de travail" />

Le code d'activation expire apres 72 heures. En cas d'expiration, cliquez a nouveau sur **Inviter** pour envoyer un nouveau code.
