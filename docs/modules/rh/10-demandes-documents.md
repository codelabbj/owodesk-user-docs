# Demandes de documents

Les employes peuvent demander des documents administratifs via le systeme.

## Demander un certificat de travail

1. Ouvrez **RH** > **Demandes de documents**.
2. Cliquez sur **Nouvelle demande**.
3. Selectionnez **Certificat de travail**.
4. Le systeme pre-remplit les informations de l'employe.
5. Cliquez sur **Soumettre**.

<DocImage src="/img/guides/rh/demande-certificat.png" alt="Demande de certificat de travail" />

## Demander une attestation de stage

1. Ouvrez **RH** > **Demandes de documents**.
2. Cliquez sur **Nouvelle demande**.
3. Selectionnez **Attestation de stage**.
4. Verifiez les dates de debut et fin de stage.
5. Cliquez sur **Soumettre**.

## Workflow de traitement

1. **Soumise** : l'employe soumet la demande.
2. **En attente** : le RH prend en charge la demande.
3. **Prete** : le document est genere et pret a etre remis.
4. **Remise** : le document a ete remis a l'employe.
5. **Rejetee** : la demande a ete refusee avec motif.

### Marquer comme pret

1. Ouvrez la demande en statut **En attente**.
2. Telechargez le document genere.
3. Cliquez sur **Marquer comme pret**.
4. Le statut passe a **Prete**.

<DocImage src="/img/guides/rh/document-pret.png" alt="Document marque comme pret" />

### Marquer comme remis

1. Ouvrez la demande en statut **Prete**.
2. Cliquez sur **Marquer comme remis**.
3. Le statut passe a **Remise** et la demande est cloturee.
