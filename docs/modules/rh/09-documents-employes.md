# Documents employes

Telechargez, organisez et recherchez les documents lies aux employes.

## Types de documents

Le systeme prend en charge les categories suivantes :

- **Contrat** : contrat de travail signe
- **CNSS** : attestation d'affiliation, carte CNSS
- **Piece d'identite** : CNI, passeport, permis de sejour
- **RIB** : releve d'identite bancaire
- **Certificat** : certificats de travail, diplomes
- **Medical** : certificats medicaux, visites

## Telecharger / uploader un document

1. Ouvrez **RH** > **Employes**.
2. Cliquez sur l'employe concerne.
3. Allez dans l'onglet **Documents**.
4. Cliquez sur **Ajouter un document**.
5. Selectionnez :
   - La **categorie** du document (contrat, CNSS, piece d'identite...)
   - Le **fichier** a telecharger (PDF, image)

<DocImage src="/img/guides/rh/upload-document.png" alt="Telechargement d'un document employe" />

6. Cliquez sur **Enregistrer**.

## Recherche par mot-cle

1. Ouvrez **RH** > **Documents employes**.
2. Utilisez la barre de recherche pour filtrer par :
   - Nom de l'employe
   - Type de document
   - Mot-cle dans le nom du fichier
3. Les resultats se mettent a jour en temps reel.
