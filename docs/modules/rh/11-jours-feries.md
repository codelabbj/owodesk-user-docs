# Jours feries

## Creer un jour ferie

1. Ouvrez **RH** > **Jours feries**.
2. Cliquez sur **Nouveau jour ferie**.
3. Saisissez :
   - Le **nom** du jour ferie (ex : Fete du Travail, Noel, Tabaski)
   - La **date** du jour ferie
4. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/rh/creer-jour-ferie.png" alt="Creation d'un jour ferie" />

## Filtrer par annee

1. Ouvrez **RH** > **Jours feries**.
2. Utilisez le filtre **Annee** pour afficher les jours feries d'une annee specifique.
3. La liste affiche le nom, la date et le jour de la semaine.

<DocImage src="/img/guides/rh/liste-jours-feries.png" alt="Liste des jours feries" />

## Impact sur les calculs

Les jours feries sont pris en compte automatiquement dans :

- **Calcul des conges** : les jours feries ne sont pas deduits du solde de conges
- **Calcul du temps de travail** : les jours feries sont considers comme des jours chomes
- **Pointage et presence** : aucun pointage n'est attendu un jour ferie
- **Jours ouvres attendus** : les jours feries sont exclus du denominateur
