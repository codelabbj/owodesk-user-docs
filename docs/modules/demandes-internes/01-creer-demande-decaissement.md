# Creer une demande interne

Les demandes internes permettent de solliciter une action ou une validation aupres d'un collaborateur. Quatre types de demandes sont disponibles.

## Types de demande

- **Decaissement** -- Sortie de fonds (remboursement, avance, paiement fournisseur)
- **Fourniture** -- Achat de materiel, fournitures de bureau ou consommables
- **RH** -- Demande liee aux ressources humaines (absence, autorisation, formation)
- **Autre** -- Demande libre sans structure predefinie

## Creer une demande de decaissement

1. Ouvrez **Demandes** > **Nouvelle demande**.
2. Dans le champ **Type**, selectionnez **Decaissement**.
3. Renseignez le **montant** (en euros ou dans la devise de l'organisation).
4. Saisissez le **motif** du decaissement (exemple : remboursement de frais de deplacement).
5. Selectionnez le **mode de paiement** dans la liste :
   - Virement bancaire
   - Cheque
   - Especes
   - Carte bancaire
6. Ajoutez des **pieces jointes** si necessaire (justificatifs, factures, notes de frais) en cliquant sur **Ajouter un fichier**.
7. Dans le champ **Destinataire**, selectionnez la personne chargee de traiter la demande.
8. Cliquez sur **Envoyer** pour soumettre la demande.

## Creer une demande de fournitures

1. Ouvrez **Demandes** > **Nouvelle demande**.
2. Selectionnez le type **Fourniture**.
3. Dans la section **Liste d'articles**, cliquez sur **Ajouter un article**.
4. Pour chaque article, renseignez :
   - Le **nom** ou la **reference** de l'article
   - La **quantite** souhaitee
   - Une **description** optionnelle (modele, couleur, format)
5. Ajoutez autant d'articles que necessaire en repetant l'etape 4.
6. Dans le champ **Destinataire**, selectionnez la personne qui validera la demande.
7. Cliquez sur **Envoyer**.

<DocImage src="/img/guides/demandes/demande-fourniture-creation.png" alt="Creation d'une demande de fournitures" />

## Creer une demande RH

1. Ouvrez **Demandes** > **Nouvelle demande**.
2. Selectionnez le type **RH**.
3. Saisissez le **motif** de la demande (exemple : demande de formation, autorisation de travail a distance).
4. Renseignez les **dates** concernees :
   - **Date de debut** de la periode
   - **Date de fin** de la periode (si applicable)
5. Ajoutez des pieces jointes si necessaire.
6. Selectionnez le **Destinataire** (generalement le responsable RH ou le superviseur).
7. Cliquez sur **Envoyer**.

<DocImage src="/img/guides/demandes/demande-rh-creation.png" alt="Creation d'une demande RH" />

## Creer une demande de type Autre

1. Ouvrez **Demandes** > **Nouvelle demande**.
2. Selectionnez le type **Autre**.
3. Saisissez un **objet** (titre court de la demande).
4. Dans le champ **Description**, detaillez votre demande en texte libre.
5. Ajoutez des pieces jointes si necessaire.
6. Selectionnez le **Destinataire** approprie.
7. Cliquez sur **Envoyer**.

<DocImage src="/img/guides/demandes/demande-autre-creation.png" alt="Creation d'une demande de type Autre" />

## Assigner un destinataire

1. Lors de la creation d'une demande, le champ **Destinataire** est obligatoire.
2. Cliquez sur le champ pour afficher la liste des collaborateurs.
3. Commencez a taper le nom d'un utilisateur pour filtrer la liste.
4. Selectionnez le destinataire souhaite.
5. La demande lui sera notifiee automatiquement apres envoi.

> Le destinataire peut etre modifie ulterieurement par un gestionnaire depuis la fiche de la demande.
