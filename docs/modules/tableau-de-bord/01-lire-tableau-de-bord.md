# Lire le tableau de bord

Le tableau de bord affiche les indicateurs cles de chaque module de l'application sur une seule page.

## Vue d'ensemble des indicateurs cles

1. A la connexion, le tableau de bord est la page d'accueil. Ouvrez-le depuis le menu **Tableau de bord**.
2. La page est organisee en **widgets** (blocs d'indicateurs) regroupes par module.
3. Chaque widget affiche les informations les plus recentes de son module.
4. Cliquez sur un widget ou sur le lien **Voir plus** pour acceder au module concerne.

<DocImage src="/img/guides/tableau-de-bord/vue-ensemble.png" alt="Vue d'ensemble du tableau de bord" />

## Module CRM

Le widget CRM affiche :
- **Nombre de contacts** total dans le CRM
- **Nombre d'entreprises** enregistrees
- **Affaires en cours** avec le montant total du pipeline
- **Activites recentes** (appels, emails, reunions)

Cliquez sur le widget pour ouvrir le module **CRM**.

## Module Facturation

Le widget Facturation affiche :
- **Revenus du mois** (montant total facture)
- **Factures impayees** (nombre et montant total)
- **Factures en retard** avec le nombre de jours de retard

Cliquez sur le widget pour acceder a la liste des **Factures**.

## Module Stock

Le widget Stock affiche :
- **Produits en alerte** (stock faible ou rupture)
- **Mouvements recents** (entrees et sorties)
- **Valeur totale du stock**

Cliquez sur un produit en alerte pour ouvrir sa fiche article.

## Module RH

Le widget RH affiche :
- **Effectifs** (nombre total d'employes)
- **Presences du jour** (employes presents, en conge, en mission)
- **Masse salariale** du mois en cours

Cliquez sur le widget pour ouvrir le module **RH**.

## Module POS (Point de Vente)

Le widget POS affiche :
- **Ventes du jour** (montant total et nombre de transactions)
- **Sessions ouvertes** (caisses en cours d'utilisation)
- **Produits les plus vendus** du jour

Cliquez sur le widget pour acceder au module **POS**.

## Module Projets

Le widget Projets affiche :
- **Taches en cours** (nombre total)
- **Taches en retard** avec leur priorite
- **Progression moyenne** des projets actifs

Cliquez sur une tache pour l'ouvrir dans le module **Projets**.

## Module E-commerce

Le widget E-commerce affiche :
- **Commandes recentes** (dernieres 24h)
- **Paniers abandonnes** (nombre et montant total)
- **Chiffre d'affaires** du jour

Cliquez sur le widget pour ouvrir le module **E-commerce**.

## Tableau de bord comptable

Le widget Comptabilite affiche :
- **Tresorerie** (solde des comptes bancaires)
- **Resultat net** du mois (revenus - depenses)
- **Echeances** a venir

Cliquez sur le widget pour ouvrir le module **Comptabilite**.

<DocImage src="/img/guides/tableau-de-bord/widget-comptable.png" alt="Widget comptable du tableau de bord" />

## Personnaliser l'affichage

1. Cliquez sur l'icone **Personnaliser** en haut a droite du tableau de bord.
2. Vous pouvez :
   - **Reorganiser** les widgets par glisser-deposer
   - **Masquer** ou **afficher** des widgets selon vos besoins
   - **Redimensionner** certains widgets
3. Cliquez sur **Enregistrer** pour conserver la configuration.
4. Chaque utilisateur peut avoir sa propre configuration du tableau de bord.

> Les donnees du tableau de bord sont mises a jour automatiquement en temps reel. Un rafraichissement manuel est possible avec le bouton **Actualiser**.
