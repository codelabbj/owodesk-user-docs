# Tableau de bord

Le Tableau de bord offre une vue d'ensemble de l'activite de votre entreprise en agregeant les indicateurs cles de chaque module.

## Indicateurs par module

- **CRM** -- Nombre de contacts, entreprises, affaires en cours
- **Facturation** -- Revenus du mois, factures impayees
- **Stock** -- Produits en alerte, mouvements recents
- **RH** -- Effectifs, presences du jour, masse salariale
- **POS** -- Ventes du jour, sessions ouvertes
- **Projets** -- Taches en cours, avancement
- **E-commerce** -- Commandes recentes, paniers abandonnes
- **Comptabilite** -- Tresorerie, resultat net
