# Zones de livraison

Definissez les zones geographiques dans lesquelles vous livrez.

## Creer une zone de livraison

1. Ouvrez **E-commerce** > **Livraison** > **Zones de livraison**.
2. Cliquez sur **Nouvelle zone**.
3. Saisissez :
   - Le **nom** de la zone (ex : "Cotonou Centre", "Porto-Novo", "Zone Nord")
   - Le **prix fixe** de livraison pour cette zone
   - Le **delai de livraison estime** :
     - **Delai minimum** (en jours)
     - **Delai maximum** (en jours)
4. Ajoutez les **villes** desservies dans cette zone (saisissez les noms, un par ligne).
5. Activez ou desactivez la zone avec l'interrupteur **Actif**.

<DocImage src="/img/guides/ecommerce/zone-livraison.png" alt="Creation d'une zone de livraison" />

6. Cliquez sur **Enregistrer**.

## Gerer les zones

Depuis la liste des zones, vous pouvez :
- Modifier le prix ou les villes
- Activer / desactiver une zone
- Supprimer une zone (sans commande en cours)
