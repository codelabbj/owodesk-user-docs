# Methodes de livraison

Configurez les differentes options de livraison proposees aux clients.

## Creer une methode de livraison

1. Ouvrez **E-commerce** > **Livraison** > **Methodes de livraison**.
2. Cliquez sur **Nouvelle methode**.
3. Saisissez le **nom** de la methode (ex : "Livraison standard", "Retrait magasin", "Express").
4. Selectionnez le **type** :
   - **Livraison** : livraison a domicile
   - **Retrait** : retrait en magasin ou point de retrait
   - **Express** : livraison acceleree
5. Activez ou desactivez la methode avec l'interrupteur **Actif**.

<DocImage src="/img/guides/ecommerce/methode-livraison.png" alt="Creation d'une methode de livraison" />

6. Cliquez sur **Enregistrer**.

La methode de livraison est visible sur la page de paiement de la boutique si elle est active.
