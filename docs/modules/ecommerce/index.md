# E-commerce (Marketplace)

Le module **E-commerce** (`/app/ecommerce`) permet de publier une boutique sur la marketplace OwoDesk, synchroniser vos produits depuis l'inventaire et traiter les commandes initiees par WhatsApp.

Dans l'application, l'en-tete du module affiche **Marketplace**. Le menu lateral reste **E-commerce**.

| Onglet dans l'app | Documentation |
|-------------------|---------------|
| **Vue d'ensemble** | [Vue d'ensemble](/modules/ecommerce/vue-ensemble) |
| **Produits** | [Produits](/modules/ecommerce/produits) |
| **Commandes initiees** | [Commandes initiees](/modules/ecommerce/commandes-initiees) |
| **Parametres** | [Parametres boutique](/modules/ecommerce/parametres-boutique) |

## Parcours type

1. [Creez votre boutique](/modules/ecommerce/parametres-boutique) (identite, contact, visuels).
2. Publiez des produits depuis l'[Inventaire](/modules/stock/produits) puis verifiez-les dans [Produits](/modules/ecommerce/produits).
3. Suivez l'activite sur la [Vue d'ensemble](/modules/ecommerce/vue-ensemble).
4. Traitez les [commandes initiees](/modules/ecommerce/commandes-initiees) en facture brouillon.

## Centre d'aide integre

Le bouton **Centre d'aide** resume les quatre sections du module.

## Prerequis

- Module **Stock** actif pour les produits et entrepots ;
- Module **Facturation** pour traiter une commande initiee en facture.
