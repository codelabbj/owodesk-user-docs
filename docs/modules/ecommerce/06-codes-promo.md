# Codes promo

Creez des codes promotionnels pour vos clients.

## Creer un code promo

1. Ouvrez **E-commerce** > **Codes promo**.
2. Cliquez sur **Nouveau code promo**.
3. Saisissez :
   - Le **code** (ex : "PROMO10", "BIENVENUE")
   - Le **type** de reduction :
     - **Pourcentage** : reduction en % du montant total (ex : 10%)
     - **Montant fixe** : reduction en montant absolu (ex : 2000 FCFA)
   - La **valeur** (10 pour 10%, ou 2000 pour 2000 FCFA)

<DocImage src="/img/guides/ecommerce/code-promo-creation.png" alt="Creation d'un code promo" />

4. Configurez les limites :
   - **Limite d'utilisations** : nombre maximum de fois que le code peut etre utilise (ex : 50)
   - **Periode de validite** :
     - Date de **debut**
     - Date de **fin**
5. Cliquez sur **Enregistrer**.

<DocImage src="/img/guides/ecommerce/code-promo-liste.png" alt="Liste des codes promo" />

Le code est actif uniquement pendant la periode de validite definie. Une fois la limite d'utilisations atteinte, le code est automatiquement desactive.
