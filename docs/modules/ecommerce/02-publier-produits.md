# Publier des produits en ligne

Rendez vos produits visibles sur la boutique e-commerce.

1. Creez d'abord les produits dans **Stock** > **Produits**.
2. Ouvrez **E-commerce** > **Produits publies**.

<DocImage src="/img/guides/ecommerce/produits-publies.png" alt="Produits publies" />

## Lier un produit a la boutique

3. Cliquez sur **Ajouter un produit**.
4. Selectionnez un **produit** depuis le catalogue de stock.
5. Configurez les options de publication :
   - **Visibilite** : active ou desactive (le produit peut etre masque)
   - **Prix personnalise** : vous pouvez definir un prix different du prix de vente standard
   - **Description personnalisee** : texte specifique pour la boutique (laisse vide pour utiliser la description du stock)
   - **Ordre d'affichage** : position du produit dans la liste (les nombres les plus petits apparaissent en premier)
6. Cliquez sur **Ajouter a la boutique**.

<DocImage src="/img/guides/ecommerce/lier-produit.png" alt="Liaison d'un produit a la boutique" />

7. Le produit apparaît sur la boutique publique une fois la visibilite activee.
