# Commenter une tache

## Ajouter un commentaire

1. Ouvrez **Projets** > ouvrez le projet > ouvrez la tache.
2. Faites defiler jusqu'a la section **Commentaires**.
3. Saisissez votre message dans la zone de texte.

<DocImage src="/img/guides/projets/commentaire-tache.png" alt="Ajout d'un commentaire" />

## Mentionner un collegue

4. Utilisez le symbole **@** suivi du **nom** du collegue (ex : "@Jean Dupont").
5. Une liste de suggestions s'affiche au fur et a mesure de la saisie.
6. Selectionnez le destinataire de la mention.

## Notifications

7. Lorsque vous mentionnez un collegue avec @, il recoit une **notification en temps reel** :
   - Par email
   - Dans l'application (centre de notifications)
8. Cliquez sur **Envoyer** pour publier le commentaire.

## Suivi des commentaires

Chaque commentaire est marque comme :
- **Non lu** pour les participants qui n'ont pas encore vu le commentaire
- **Lu** apres consultation

Le nombre de commentaires non lus est affiche sur la tache dans la liste.
