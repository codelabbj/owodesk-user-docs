# Creer une tache

1. Ouvrez **Projets** > ouvrez le projet concerne.
2. Cliquez sur l'onglet **Taches** > **Nouvelle tache**.
3. Saisissez :
   - Le **titre** de la tache
   - La **description** (optionnelle)
   - La **priorite** : basse, moyenne, haute, critique
   - Le **statut** : a faire, en cours, terminee
   - La **date d'echeance**
4. Renseignez les dates reelles (optionnelles) :
   - **Date de debut reelle**
   - **Date de fin reelle**

<DocImage src="/img/guides/projets/creer-tache.png" alt="Creation d'une tache" />

## Assignation

5. Dans le champ **Assigner a**, selectionnez un ou plusieurs **employes**.
6. Chaque employe assigne recevra une notification.

## Tags

7. Ajoutez des **tags** (etiquettes colorees) pour categoriser la tache :
   - Exemple : "urgent", "bug", "design", "dev"
   - Les tags sont visibles dans la liste des taches

<DocImage src="/img/guides/projets/tache-tags.png" alt="Tags colores sur une tache" />

## Mode strict

8. Activez le **mode strict** si necessaire :
   - En mode strict, seul le responsable de la tache peut la modifier
   - Les autres utilisateurs peuvent la voir mais pas l'editer

9. Cliquez sur **Enregistrer**.
