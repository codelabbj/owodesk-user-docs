# Analytics du projet

Consultez les indicateurs de performance d'un projet.

1. Ouvrez **Projets** > ouvrez le projet concerne.
2. Cliquez sur l'onglet **Analytics**.

<DocImage src="/img/guides/projets/analytics-projet.png" alt="Vue analytique d'un projet" />

## Budget vs cout reel

La vue affiche un comparatif entre :
- **Budget previsionnel** : montant alloue au demarrage
- **Cout reel** : cumul des depenses effectuees (taches, ressources, achats)
- **Ecart** : difference entre le budget et le cout reel

<DocImage src="/img/guides/projets/budget-vs-cout.png" alt="Comparaison budget vs cout reel" />

## Marge

La marge est calculee automatiquement :
- **Marge (montant)** : budget previsionnel - cout reel
- **Marge (pourcentage)** : (marge / budget previsionnel) x 100

## Total facture

Si le projet est lie a des factures (via le CRM ou le module Facturation), le **total facture** est affiche :

- Montant total facture au client
- Comparaison avec le budget previsionnel
- Rentabilite du projet

<DocImage src="/img/guides/projets/marge-projet.png" alt="Marge et total facture du projet" />
