# Ouvrir une session de caisse

Avant toute vente, une session de caisse doit etre ouverte.

1. Ouvrez **Point de vente** > **Sessions de caisse**.
2. Cliquez sur **Ouvrir une session**.
3. Selectionnez :
   - **L'entrepot** / la caisse concerne(e)
   - Le **fonds de caisse initial** (montant en especes en caisse au demarrage)
4. Ajoutez une **note d'ouverture** (optionnelle, ex : "Ouverture equipe matin").
5. Validez. Un **numero de session** est genere automatiquement (S-XXXXX).
6. La session passe au statut **Ouverte**.

<DocImage src="/img/guides/pos/session-ouverture.png" alt="Ouverture d'une session de caisse" />

## Verification

Le systeme verifie qu'aucune autre session n'est deja ouverte pour le meme entrepôt. Si une session est deja active, vous devez la fermer avant d'en ouvrir une nouvelle.

Sans session ouverte, l'encaissement est bloque.
