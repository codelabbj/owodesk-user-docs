# Consulter le rapport Z

Visualisez le rapport Z d'une session de caisse terminee.

1. Ouvrez **Point de vente** > **Sessions de caisse**.
2. Cliquez sur une session **Fermee**.
3. Cliquez sur **Rapport Z**.

<DocImage src="/img/guides/pos/rapport-z-consultation.png" alt="Consultation du rapport Z" />

## Informations affichees

Le rapport Z contient :

- **Total des ventes** de la session (montant cumule)
- **Nombre de ventes** realisees pendant la session
- **Ventilation par mode de paiement** :
  - Montant en especes
  - Montant par carte bancaire
  - Montant par mobile money
- **Top 10 produits** les plus vendus (produit, quantite, montant)
- **Ventes annulees** : liste des annulations avec motif

<DocImage src="/img/guides/pos/rapport-z-detail.png" alt="Detail du rapport Z" />

## Renvoyer le rapport par email

1. Depuis la vue du rapport Z, cliquez sur **Envoyer par email**.
2. Saisissez l'**adresse email** de destination.
3. Cliquez sur **Envoyer**.
4. Le rapport est envoye au format PDF.
