# Creer une vente

Creez une vente en mode brouillon avant de l'encaisser.

1. Ouvrez **Point de vente** > **Ventes**.
2. Cliquez sur **Nouvelle vente**.
3. La vente est creee avec le statut **Brouillon**.

## Ajouter des articles

1. Dans la vente ouverte, cliquez sur **Ajouter un article**.
2. Recherchez un **produit** dans le catalogue (stock).
3. Si le produit a des **variantes** (taille, couleur), selectionnez la variante.
4. Saisissez la **quantite** vendue.
5. Le **prix unitaire** s'affiche automatiquement selon le tarif standard ou les **paliers de prix** configures.
6. La **TVA** est calculee automatiquement selon le taux applicable au produit.
7. Cliquez sur **Ajouter** pour ajouter l'article a la vente.

<DocImage src="/img/guides/pos/ajouter-article.png" alt="Ajout d'un article a la vente" />

8. Repetez l'operation pour chaque article.

## Modifier une vente en brouillon

Tant que la vente est en statut **Brouillon**, vous pouvez :
- Modifier les quantites
- Supprimer des articles
- Ajouter de nouveaux articles
- Appliquer des remises
