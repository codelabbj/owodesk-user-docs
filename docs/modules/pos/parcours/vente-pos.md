# Vente au comptoir (POS)

Parcours rapide pour une vente en magasin.

## Étapes

1. Ouvrir une **session** POS

<DocImage src="/img/guides/pos/session-ouverture.png" alt="Sessions de caisse — ouverture de session" />

2. Ajouter des lignes (produit, quantité, prix)
3. Choisir le mode de paiement
4. Imprimer le ticket

<DocImage src="/img/guides/pos/caisse-encaissement.png" alt="Écran d'encaissement POS" />

5. Le stock est décrémenté automatiquement
6. Fermer la session en fin de journée

<DocImage src="/img/guides/pos/session-fermeture.png" alt="Sessions de caisse — clôture" />
